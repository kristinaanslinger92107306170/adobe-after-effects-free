### Создание простой анимации

1. Импортируйте изображение или видео (Файл → Импорт)
2. Перетащите файл на значок "Новая композиция"
3. Выделите слой на временной шкале
4. Нажмите "P" — откроется параметр "Положение"
5. Нажмите на значок секундомера — создастся ключевой кадр
6. Переместите ползунок времени вперёд
7. Передвиньте объект в новое положение
8. Нажмите пробел для просмотра анимации

### Добавление текста

1. Выберите инструмент **"Текст"** (клавиша Ctrl+T)
2. Нажмите на область композиции и введите текст
3. Выделите текстовый слой
4. В панели **"Символ"** настройте шрифт, размер, цвет
5. Анимируйте текст через параметры "Положение", "Масштаб", "Поворот"

### Популярные эффекты

| Эффект | Назначение |
|--------|-----------|
| **Свечение** | Glow — добавляет сияние |
| **Размытие в движении** | Motion Blur — эффект скорости |
| **Цветокоррекция** | Curves, Levels — настройка цвета |
| **Частицы** | Particle World — снег, дым, искры |
| **Стабилизация** | Warp Stabilizer — убирает тряску |

### Трекинг движения

1. Выберите слой с видео
2. Анимация → Трекинг движения
3. Выберите точку для отслеживания
4. Нажмите "Анализировать"
5. Примените данные трекинга к целевому слою
6. Объект будет двигаться вместе с видео

### Экспорт готового видео

1. Нажмите **Композиция** → **Добавить в очередь рендеринга**
2. Нажмите на "Потеря без сжатия" рядом с "Выходной модуль"
3. Выберите формат: **QuickTime** или **H.264**
4. Нажмите на "Установки" для выбора места сохранения
5. Нажмите **"Рендерить"**
6. Готово — видео сохранено
